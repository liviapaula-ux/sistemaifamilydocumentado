01 Processamento e Higienização de Dados

INPUT

Captação

Lê o Google Form e filtra inscrições pós-Março/26.

CLEANING

Deduplicação

Compara e-mails para evitar registros repetidos na base.

MAPPING

Slack Lookup

Converte e-mail corporativo em ID interno do Slack.

SAFETY

Anti-Spam

Consulta a planilha 'Controle' para não repetir mensagens já enviadas.

03 Sincronização de Dados (Estratégia Multibases)

O Agente iFamily 2.0 não depende de uma única base. Ele sincroniza informações de múltiplas frentes para garantir precisão:

Base de Saúde

Cruza dados com os Checkpoints de Saúde 1 e 2 para validar participação.

Assistência Social

Verifica o status de agendamento com o assistente social do projeto.

Kits e Logística

Integração com a planilha de Controle de Kits para avisar o colaborador do envio.
